# ONTHI - Bai tap TH Lap trinh OOP

Code C++ giai cac bai trong file **TH LAP TRINH OOP (1).pdf**. Tat ca da bien dich & chay thu OK voi `g++ -std=c++17`.

## Cach bien dich & chay
```bash
g++ -std=c++17 Tuan1/bai1_matrix_thanhphan.cpp -o bai1
./bai1
```
